This project is the fourth homework from the GoIT HTML/CSS course.  
It contains continuation of the WebStudio landing page with improved layout, responsive design, and enhanced styles.

## 🧰 Technologies

- HTML5
- CSS3
- Git
- GitHub

## 🚀 Features

- Semantic HTML structure
- Advanced CSS layout
- Responsive design for different screen sizes
- Navigation improvements
- Sections: Hero, Features, Team, Portfolio, Footer

Open the `index.html` file in your browser to view the project.
